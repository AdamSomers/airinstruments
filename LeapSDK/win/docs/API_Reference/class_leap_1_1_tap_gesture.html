Detailed Description

The TapGesture class represents a tapping gesture by a finger or tool.

A tap gesture is recognized when the tip of a finger rotates down toward the palm and then springs back to approximately the original postion, as if tapping. The tapping finger must pause briefly before beginning the tap.

Important: To use tap gestures in your application, you must enable recognition of the tap gesture. You can enable recognition with: 

TapGesture::enable();

Tap gestures are discrete. The TapGesture object representing a tap always has the state, STATE_STOP. Only one TapGesture object is created for each tap gesture recognized.

Vector Leap::TapGesture::direction ( ) const

The current direction of finger tip motion.

At the start of the tap gesture, the direction points in the direction of the tap. At the end of the tap gesture, the direction will either point toward the original finger tip position or it will be a zero-vector, which indicates that finger movement stopped before returning to the starting point.

Returns
Vector A unit direction vector if the finger tip is moving; otherwise, a zero-vector.
static void Leap::TapGesture::enable ( bool  enable = true)
inlinestatic

Enables or disables reporting of tap gestures.

By default, tap gestures are disabled and are never reported in the frame gesture list.

As a performance optimization, only enable recognition for the types of movements that you use in your application.

Parameters
enableTrue, to enable the specified gesture type; False, to disable.
