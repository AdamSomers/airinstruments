Detailed Description

The ZoomGesture represents a zooming/scaling gesture performed by two pointable objects.

The Leap recognizes a zoom gesture when two pointable objects move toward or away from each other. (Currently, the gesture is only recognized when there are two and only two pointable objects in view.)

Important: To use zoom gestures in your application, you must enable recognition of the zoom gesture. You can enable recognition with: 

ZoomGesture::setListenFor( true );

Zoom gestures are continuous. As long as the gesture continues, a ZoomGesture object is present in the frame gesture list. The objects scaleFactor() reports the current zoom ratio.

Note that you can also derive zoom factors from the frame and hand scale motions.

float Leap::ZoomGesture::scaleFactor ( ) const

The current scale factor.

The zoom or scale factor is set to 1.0 at the start of the gesture. As the pointable objects move closer the scale factor gets smaller and approaches zero. As they move farther apart, the scale factor gets larger.

Returns
float A positive number indicating the current zoom multiplier.
static void Leap::ZoomGesture::setListenFor ( bool  listenFor)
inlinestatic

Enables or disables recognition and reporting of zoom gestures.

By default, zoom gestures are disabled and not reported in the frame gesture list.

As a performance optimization, only enable recognition for the types of gestures that you use in your application.

Parameters
listenForTrue, to enable the zoom gesture type; False, to disable.
